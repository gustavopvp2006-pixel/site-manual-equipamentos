# 📡 Central Técnica - Manual de Equipamentos

Este projeto é um site desenvolvido com o objetivo de centralizar informações técnicas e manuais de equipamentos utilizados no dia a dia de suporte em telecomunicações.

## 🔎 Sobre o Projeto

A **Central Técnica** foi criada para facilitar o acesso rápido a informações importantes sobre equipamentos como ONUs, OLTs e sistemas utilizados no suporte técnico.

O site possui uma interface simples, moderna e intuitiva, permitindo que técnicos encontrem facilmente orientações e instruções necessárias para atendimento ao cliente.

## 🚀 Funcionalidades

* 📘 Exibição de manuais técnicos
* 🔍 Organização de conteúdos por tipo de equipamento
* ⚙️ Auxílio no suporte técnico diário
* 💻 Interface responsiva e de fácil navegação
* 🎯 Foco em agilidade no atendimento

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome (ícones)
* Google Fonts

## 📂 Objetivo

Este projeto foi desenvolvido com foco em:

* Melhorar a produtividade no suporte técnico
* Reduzir o tempo de atendimento
* Centralizar informações importantes
* Servir como base de consulta rápida para técnicos
